1. 교과목 개요

본 과정은 순수 ROS2 또는 로보틱스 구현 강의가 아니라, 국방 AI 환경에서 자율무인체계와 드론 군집이 직면하는 보안 위협을 분석하고 공격·방어 관점에서 실험하는 프로젝트 기반 집중 교육과정이다.

학습자는 최신 논문과 사례를 바탕으로 위협 모델을 이해하고, 시뮬레이션 기반 환경에서 Red Team / Blue Team 시나리오를 설계하며, 방산 AI 보안 연구 주제를 발표 가능한 형태로 발전시킨다.

Defense AI Security Drone Swarm Autonomous Systems Cyber-Physical Security Red Team / Blue Team ROS2 + Gazebo + PX4 Simulation

2. 연구형 학습 구조

Research-oriented flipped learning
논문 기반 개념 학습에서 공격·방어 시뮬레이션 프로젝트까지

7. 학습 일정

단계 방식 주제 학습 방향 시간

8. 온라인 학습 안내

Flipped Learning

  • 짧은 강의 영상으로 핵심 개념 사전 학습
  • 최신 논문과 사례 중심의 연구형 학습
  • AI/drone security concepts와 toy problem 수행

사전 발표 준비

학습자는 국방 AI, 드론 보안, 자율무인체계와 관련된 연구 주제 또는 논문을 선정하고, 오프라인 실습이 시작되기 전에 짧은 발표 제안서를 준비한다.
